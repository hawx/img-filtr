# img-filtr

Port of [straup/filtr][filtr] to use [hawx/img][img].

- [x] brdl
- [x] dazd
- [x] dthr
- [x] dthrpxl (equivalent to `img pxl | img dthr`)
- [x] edwn
- [x] heathr
- [x] postcrd
- [x] postr
- [x] pxl (in img)
- [x] pxldthr (equivalent to `img dthr | img pxl`)
- [x] rockstr


May possibly look at adding [rcrowley/curvr][curvr] too.
